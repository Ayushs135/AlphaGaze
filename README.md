# AlphaGaze

AlphaGaze is a multimodal financial forecasting system that combines
time-series analysis with sentiment-aware NLP to model market volatility.

## Tech Stack
- Python
- Facebook Prophet
- yfinance
- BERT / FinBERT
- Jupyter Notebook


## Dataset

### Financial News Data
- **Source:** Indian Financial News Dataset (Zenodo)
- **Link:** https://zenodo.org/records/16991843
- **Description:** A publicly available dataset containing Indian financial news articles used for sentiment analysis with FinBERT.

### Market Data
- **Source:** Yahoo Finance
- **Accessed via:** yfinance
- **Asset:** NIFTY 50 Index (^NSEI)
